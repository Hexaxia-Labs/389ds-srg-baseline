# Roadmap

This catalog is the single source of truth. Later phases generate automation
from the same control front-matter - no re-authoring.

| Phase | Deliverable | Status |
| --- | --- | --- |
| 1 | Control schema, 43 authored controls, docs site, CI | in progress |
| 2 | Bash audit script + Ansible remediation role | planned |
| 3 | OpenSCAP XCCDF/OVAL content (`oscap`-scannable) | planned |
| 4 | Upstream contribution to ComplianceAsCode | planned |
