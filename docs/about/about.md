# About this project

389DS SRG Baseline is built and maintained by Aaron Lamb at
[Hexaxia Technologies](https://www.hexaxia.tech), and released as open source
through the [Hexaxia Labs](https://github.com/Hexaxia-Labs) GitHub organization.

It began as internal hardening work for directory infrastructure and grew into a
public, reusable baseline. It is **alpha / draft** - the control commands have not
yet been validated against a live 389 Directory Server and have not been
independently reviewed. See the [disclaimer](../index.md) on the home page.

## Hexaxia Technologies

[Hexaxia Technologies](https://www.hexaxia.tech) is an infrastructure and services
company doing managed IT, consulting, and security and compliance work. Hardening
systems like 389 Directory Server is part of the day-to-day, and this baseline is
one of the things that came out of that work.

## Maintainer

Aaron Lamb is the founder of Hexaxia Technologies, with 20+ years of experience in
IT, Linux, and systems infrastructure. More at
[www.hexaxia.tech](https://www.hexaxia.tech).
